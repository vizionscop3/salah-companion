## **Product Strategy**

### **Target Users**

1. **New Muslims / Reverts**
   - Adults who recently embraced Islam
   - No prior Arabic exposure
   - High motivation, significant knowledge gaps

2. **Born Muslims Seeking Depth**
   - Raised Muslim but never taught properly
   - Can recite but don't understand meaning
   - Desire informed worship

3. **Muslims with Learning Differences**
   - ADHD, dyslexia, cognitive challenges
   - Need alternative engagement methods
   - Benefit from gamification

## **User Stories**

### **Epic 1: Prayer Guidance**
- As a new Muslim, I want step-by-step guidance through prayer so I can perform Salah correctly
- As a user, I want to understand what I'm reciting so I can connect spiritually
- As a user with ADHD, I want bite-sized learning chunks so I can focus and progress

### **Epic 2: Azan System**
- As a user, I want beautiful Azan notifications so I never miss prayer time
- As a new Muslim, I want to learn the meaning and response to Azan so I can participate properly
- As a user, I want customizable Azan settings so it fits my lifestyle

### **Epic 3: Recitation Practice**
- As a user, I want to practice recitation and get feedback so I can improve my pronunciation
- As a user, I want to see my progress over time so I can track improvement
- As a user, I want tajweed analysis so I can recite correctly

### **Epic 4: Prayer Times**
- As a user, I want accurate prayer times for my location so I never miss a prayer
- As a user, I want to understand why prayers are at these times so I can appreciate their significance
- As a traveler, I want automatic time zone adjustment so prayer times stay accurate

### **Epic 5: Islamic Calendar**
- As a new Muslim, I want to know when Islamic holidays are so I can participate
- As a user, I want to learn the significance and conduct for each holiday so I can observe properly
- As a user, I want countdowns to important dates so I can prepare

### **Epic 6: Gamification**
- As a user with learning differences, I want achievements and rewards so learning feels accessible
- As a user, I want to track my consistency so I can build habits
- As a user, I want multiple pathways to success so I can learn in my own way
